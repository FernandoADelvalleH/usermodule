REQUERIMIENTOS

	1.- Tener instalado algun servidor de php como Xampp, Lampp o Wampp.
	3.- Instalar algun navegador Web como Chrome, Edge (Explorer), Safari, Mozilla o similar.


INTALACIÓN DE APLICACIÓN WEB

	1.- Descargar proyecto UserModule (LINK).
	2.- Descomprimir y mover carpeta de proyecto a la direccion del "htdocs" de cualqueir servidor de php (ejemplo Lampp, Xampp, Wamp...).
	3.- ingresar a phpmyadmin y crear una nueva ase de datos con el nombre "ejemplos".
	4.- Ingresar al menú de importacion una vez creada y seleccionada la base de datos "ejemplos".
	5.- En la sección "Archivo a importar" darclic al boton "Seleccionar archivo".
	6.- En el cuadro de dialogo del exporador de archivos, dentro de la carpeta del proyecto "usermodule", localizar archivo de la base de datos con el nombre "ejemplos.sql" y seleccionar.
	7.- Dar clic en el boton "Abrir".
	8.- Esperar que se cargue la informacion a la base de datos.
